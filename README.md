# rate-watch-bot

Telegram-бот, который следит за курсом криптовалют (через CoinGecko, без API-ключа) и пишет тебе, когда цена пробивает заданный уровень. Сделал для себя — устал каждые полчаса открывать вкладку с курсом.

## Команды

```
/watch bitcoin above 70000   — сообщить, когда BTC станет выше $70,000
/watch ethereum below 2500   — сообщить, когда ETH станет ниже $2,500
/list                        — показать свои активные алерты
```

## Запуск

```bash
pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
python -m ratewatch.bot
```

Бот проверяет цены раз в 60 секунд (настраивается через `POLL_INTERVAL_SECONDS`). Алерты хранятся в `alerts.json` — если процесс перезапустится, подписки не потеряются.

## Структура

```
ratewatch/
  prices.py   — запрос цены к CoinGecko
  alerts.py   — модель алерта + файловое хранилище (без БД)
  bot.py      — обработчики команд и цикл проверки
```

## Тесты

Логика алертов покрыта тестами (без сетевых запросов — CoinGecko не мокается, а просто не участвует в этих тестах):

```bash
pip install requests pytest
pytest tests/ -v
```

## Ограничения

- Хранилище — JSON-файл, подходит для личного использования или небольшого числа пользователей, не для продакшена с нагрузкой.
- Один алерт срабатывает один раз (`triggered = True`), повторно не шлётся — если нужно следить постоянно, добавляй новый `/watch`.
- Цена берётся с округлением до momento запроса — задержка в пределах `POLL_INTERVAL_SECONDS`.
